## GUADELOUPE

### 1. CTM Deher (Trois-Rivières ↔ Les Saintes)
- **Site web** : www.ctmdeher.com
- **Téléphone** : 0590 92 06 39
- **Routes** : Trois-Rivières → Terre-de-Haut / Terre-de-Bas
- **Tarifs** :
  - Aller simple : 22€
  - Aller-retour : 45-55€
- **Fréquence** : 7j/7, 10+ rotations/jour
- **Flotte** : 4 vedettes (181-316 places)
- **Durée** : 20-25 min
- **Services** : Transport piétons uniquement

### 2. Karu'Ferry (Trois-Rivières ↔ Les Saintes)
- **Site web** : www.karuferry.com
- **Routes** : Trois-Rivières → Terre-de-Haut / Terre-de-Bas, Bouillante → Les Saintes
- **Tarifs** : 21€ AR
- **Fréquence** : 7j/7, 8h-18h15
- **Particularité** : Propulsion solaire - navire éco-responsable
- **Durée** : 25 min
- **Navette inter-Saintes** : GRATUITE

### 3. Val'Ferry
- **Site web** : www.valferry.fr
- **Téléphone** : 0590 91 45 15
- **Routes** :
  - Trois-Rivières → Les Saintes
  - Pointe-à-Pitre → Marie-Galante
  - Pointe-à-Pitre → Fort-de-France
- **Tarifs** :
  - Les Saintes : 35€ AR
  - Marie-Galante : 42€ AR (38,80€ web)
  - Martinique : 82-118€
- **Horaires Marie-Galante** :
  - Lun-Sam : 7h45, 12h30, 17h
  - Dim : 7h45, 16h45, 19h15

### 4. FRS Express
- **Site web** : www.frs-express.com
- **Téléphone** : 0825 35 90 00
- **Routes** :
  - Pointe-à-Pitre ↔ Marie-Galante
  - Pointe-à-Pitre ↔ Les Saintes
  - Pointe-à-Pitre ↔ Fort-de-France (via Dominique)
  - Pointe-à-Pitre ↔ Dominique

  - Marie-Galante : à partir de 29€ AR

- **Horaires Marie-Galante** : 8h15, 17h15, 19h30 (dim)

### 5. Comadile
- **Site web** : www.comadile.fr
- **Téléphone** : 0590 30 02 10
- **Routes** :
  - Saint-François ↔ Marie-Galante (St-Louis)
  - Saint-François → La Désirade

  - Marie-Galante : 25€ AS
  - La Désirade : 30€ AS

### 6. Navette Beatrix
- **Routes** : Trois-Rivières ↔ Les Saintes (inter-îles)
- **Tarifs** : 23€ (Terre-de-Haut), 21€ (Terre-de-Bas), 10€ (inter-Saintes)

### 7. Jeans for Freedom
- **Routes** : Pointe-à-Pitre → Les Saintes
